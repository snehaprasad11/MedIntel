# AI Insights Engine

## Overview
Generates natural language insights from structured hospital data.

---

## Components

### 1. Analytics Engine
- KPI aggregation
- statistical summaries

### 2. Insight Engine
- Converts KPIs → narratives

### 3. Recommendation Engine
- Suggests operational improvements

### 4. LLM Provider
- Connects to OpenAI API

---

## Output Examples
- "Department A shows 20% higher load than average"
- "Doctor utilization is uneven across departments"
